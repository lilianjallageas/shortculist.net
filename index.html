Mac Shortcuts

Essentials:

Shortcut command Action
⌘-C Copy the selected element
⌘-X Cut the selected element
⌘-V Paste the selected element
⌘-A Select all the elements in the window
⌘-S Saves the active document
⌘-Z Undo the previous command
⌘-⇧-Z Redo the previous command
⌘-F Find
⌘-Tab Switch to the most recent used application
⌘-H Hide the windows of the application currently used
⌘-M Minimize the window currently used
⌘-W Closes window currently used
⌘-Q Quits the application currently used

Screenshots on Mac:

Shortcut command Action
⌘-⇧-3Takes a screenshot and saves it into a file on the desktop
⌘-⇧-4Takes a screenshot of a selected area and saves it into a file on the desktop
⌘-Ctrl-⇧-3Takes a screenshot and saves it in the clipboard (for magicians only)
⌘-Ctrl-⇧-4Takes a screenshot of a selected area and saves it in the clipboard (for magicians only)

Chrome on Mac:

Shortcut command Action
⌘-NOpens a new window
⌘-TOpens a new Tab
⌘-WCloses the active Tab
⌘-⇧-TOpens the last closed Tab
⌘-RReloads the page
⌘-⇧-NOpens a new 'Incognito' window
